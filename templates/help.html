Copia de seguridad base de datos.

Migueautos cuenta con una base de datos en la cual se guardan los registros de todas las tablas y esos datos(registros) son de suma importancia, por eso nuestro software cuenta con una copia de seguridad para que estos datos se puedan mantener a salvo y a la mano, si desea ir hacer una copia de seguridad tienes que seguir los siguientes pasos:

  • Ingresar a este link : Copia de seguridad
  • Selecciona las casillas de las que deseas realizar copia de seguridad
  • Hacer click en enviar(desde ese momento ya se creo No hacer click mas de una vez :) )
  • Seleccionar la opción de copia de seguridad
